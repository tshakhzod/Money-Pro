# 💰 Money Pro — Твой финансовый помощник

**Money Pro** — это простое и надежное приложение для учета личных финансов, контроля доходов, расходов и долгов. Разработано специально для удобного управления бюджетом в Узбекистане.

---

### 🌟 Что умеет Money Pro?
* **Учет в один клик:** Быстро добавляй доходы и расходы.
* **Контроль долгов:** Отдельный раздел для записи тех, кто должен тебе, и тех, кому должен ты.
* **Валюта UZS:** Полная поддержка узбекского сума.
* **Экспорт в Excel:** Выгружай свои отчеты в форматы **.xlsx** и **.csv** для глубокого анализа.
* **Полная приватность:** Все данные хранятся только на твоем телефоне, без интернета и серверов.
* **Защита ПИН-кодом:** Твои финансы под замком.

---

### 📥 Как скачать и установить?

1. Перейди в раздел **[Releases](https://github.com/tshakhzod/Money-Pro/releases)** (справа на этой странице).
2. Скачай файл с расширением **.apk** (например, `MoneyPro_v1.0.4.apk`).
3. Открой скачанный файл на своем Android-смартфоне.
4. Если телефон спросит разрешение на установку из неизвестных источников — разреши (это стандартная процедура для файлов не из Google Play).
